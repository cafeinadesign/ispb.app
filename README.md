# ispb.app

Catálogo amigável de participantes Pix/SPI para pessoas que querem entender quem opera no ecossistema financeiro brasileiro e onde podem existir oportunidades.

## Scripts

```bash
npm run dev
npm run typecheck
npm run validate:data
npm run build
npm run preview
```

O site é estático, usa Vite + Vanilla TypeScript e consome `@thiagoprazeres/ispb-participants`.
